# Privacy Policy - UBF Appbutik

**Last updated:** February 10, 2026

## Overview

UBF Appbutik is a Chrome extension developed for Utbildningsförvaltningen Göteborgs Stad (UBF) to provide students and staff with easy access to approved Chrome extensions.

## What data is collected?

The extension collects the following information:

**From Google Workspace:**
- User's email address (for authentication)
- User's name (for display purposes)
- User's profile picture (for display purposes)

## How is the data used?

Data is used **only** for:
- Authenticating users via Google Workspace
- Displaying user profile in the extension
- Verifying organization membership

## Where is data stored?

- **Local cache:** Authentication status is cached locally in the browser
- **No server storage:** No personal data is stored on external servers

## Third parties

The extension communicates with the following third-party services:

| Service | Purpose | Data sent |
|---------|---------|-----------|
| **Google Identity API** | Authentication | Email address |

All communication is encrypted via HTTPS.

## Permissions

The extension requests the following permissions:
- **identity:** To authenticate with Google Workspace
- **storage:** For local caching of authentication status and preferences
- **sidePanel:** To display the app store interface in the Chrome side panel

## Data retention

- Authentication cache expires after browser session ends
- No personal data is permanently stored

## Contact

For questions about this privacy policy, contact Utbildningsförvaltningen Göteborgs Stad.

## Changes

We may update this policy as needed. Changes will be published on this page.
